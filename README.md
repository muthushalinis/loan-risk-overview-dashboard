# NBFC Loan Risk Overview Dashboard

This project is an Excel-based dashboard created to analyse loan risk for an NBFC-style dataset.

It covers:
- Total vs medium-risk loan exposure
- Loan count and amount by risk category
- Risk distribution across loan types
- Top cities contributing to medium-risk loans

Tools used:
- Excel
- Power Query
- Pivot Tables & Charts
- Slicers

Note: Risk classification is derived from the Notes field, which is available for ~28% of the total records.
